# **Explanation**
Hashtables are used to store key-value pairs.
Uses Hash function which is Detereministics
**Applications**

1. Spell Checkers - a word lookup
2. Dictionaries 
3. Compilers - lookup address of functions and variab;es
4. In Code Editors

**Implementations**

* Java -> HashMap
* Javascript -> Object
* Python,C# -> Dictionaries

**Operations**

* Insert O(1)
* Lookup O(1)
* Delete O(1)
